# MineDevTools 使い方

Minecraft 開発向けのVS Code 拡張機能です。  
このドキュメントでは、基本的な使い方とコマンドの概要を説明します。

## 使い始める

1. VS Code のコマンドパレットを開きます。  
   macOS: `Cmd+Shift+P` / Windows & Linux: `Ctrl+Shift+P`
2. コマンド名を入力して実行します。

## コマンド一覧

- `設定ファイル作成` (`extension.createSetting`)  
  設定ファイルを生成します。
- `Show mc dir Preview` (`extension.showMcDirPreview`)  
  Minecraft ディレクトリの内容をプレビュー表示します。
- `zipfile` (`extension.compressResourcePack`)  
  リソースパックを zip に圧縮します。
- `translate` (`extension.translate`)  
  翻訳機能を実行します。
- `combine` (`extension.combine`)  
  ファイルの結合処理を行います。
- `encoder` (`extension.encoder`)  
  エンコード処理を行います。
- `decoder` (`extension.decoder`)  
  デコード処理を行います。
- `converter` (`extension.converter`)  
  変換処理を行います。
